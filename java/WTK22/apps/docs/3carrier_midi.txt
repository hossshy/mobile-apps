今日自分で実践して得られた結果ですので、少しでも参考になれば幸いです。
以前日記でルルさん、GA-LUCKさんに色々と頂いた情報をベースにしています。
勝手ながら本稿に適宜引用していますがご了承の程お願い致します。

-- ルルさん情報 --
１．SMD
２．SMAF(mmf)
３．SMAF/Phrase(spf)
１をアプリで使うことはほとんどねぇっす。
２に関しては、PsmPlayerでも吐き出せますが、
複数の音声が鳴らせない・・・。
３は、複数は可能なんですが、１トラック（１ファイル）に対する
チャンネル数が４とか？８とか？（記憶があいまい・・・）で少ないので、
ＢＧＭでドラムとかを入れるときは、
２トラック（２ファイル）用意して、同時再生させます。
--
-- GA-LUCKさん情報 --
MA3：もっとも再生される可能性が高いがドラム音などが弱い。
MA5：ドラム音など強化されたバージョン
MA7：３Dサウンド？あまりわかってません；；
で出来るだけMA３で変換しておけば、どの機種でも再生されますのでオススメです。ただドラムやパーカション音などはやっぱりMA5には負けます。
--
-- 風舞意見 --
個人的に音質よりも機種を少しでも多く救う方が有用だと思っているのでMA3固定で良いと思います。
MA3はツール起動後 Option->Preference->AT Mode でL1(MA-3)を選択すること。
マジで4チャネル毎にファイルが分かれる（0.spf,0_2.spf,0_3.spf）ので注意。
同期のさせ方とかはMEXA SDKのjavadocを読むと理解可能。
