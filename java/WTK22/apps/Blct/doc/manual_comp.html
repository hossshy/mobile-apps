[勲章[黄/青]]
最終ボスまで倒すと、そのモードの機体セレクト画面に勲章が表示されます。
勲章は2種類存在し、通常プレイで黄色の勲章
プレイ中一度もトランシードを使用せずにクリアすると青色の勲章が獲得できます。
気軽に片手でプレイしたい時、トランシードを使わずにプレイしてみてはいかがでしょうか。
当然トランシードを使わない方が難しいです。


[Lv.段位認定]
それぞれのモード＆機体のプレイ記録により、そのモードの機体セレクト画面に段位が表示されます。
基準は、より難しいモードを、より高得点で進めることです。


[隠し要素]
本アプリには隠し要素が3つ存在します。

・サウンドトラック
　どのモードでも構わないので最終ステージまで到達
・3機目出現
　どのモードでも構わないので最終ボスまで撃破
・Chaosモード
　Hardモードで5面まで到達（リトライ可）
